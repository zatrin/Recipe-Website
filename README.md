# Recipe-Website

This is a simple recipe website made using HTML and CSS. It shows different food recipes in a clean and easy-to-use layout.

The goal of this project is to practice frontend development and create a responsive website that works on both mobile and desktop.

# Features
Simple and clean UI
Recipe display with images and details
Easy navigation between sections
Responsive design for all devices
Built using only frontend technologies
# Technologies Used
HTML
CSS

# Project Purpose

This project was built for learning frontend development and improving UI design skills.
